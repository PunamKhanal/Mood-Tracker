A modern, responsive web application built with Vanilla JavaScript, HTML, and CSS to track daily moods and activities.

## ✨ Features
- **Mood Selection:** Choose from 5 different emojis to record your emotional state.
- **Activity Logging:** Record 1-3 short highlights of your day.
- **Data Persistence:** Uses `localStorage` so data remains after page reload.
- **Dynamic History:** View, filter, and delete past entries instantly.
- **Statistics:** Real-time tracking of total entries and most frequent mood.

## 🧠 Reflection
For this project, I developed a Mood Tracker to practice DOM manipulation and event handling. The biggest technical challenge was ensuring the data persisted after a browser refresh. I solved this by using `JSON.stringify()` and `JSON.parse()` with the browser's **localStorage**. 

I am proud of the modern "Glassmorphism" design and the smooth blue gradient which makes the app feel like a premium mobile tool. In the future, I want to add a "Dark Mode" toggle and a chart to visualize mood trends over a full month.
